Classic Minesweeper Game

Each field has a number representing the mines surrounding it.
Sweep all the fields that dont have mines.


Comes with 3 modes:
	*Easy: 9x9 - 10 mines.
	*Medium: 16x16 - 40 mines.
	*Hard: 30x16 - 100 mines.
